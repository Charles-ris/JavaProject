Comment installer la distribution source de cette application ?

Requis

  1. JSE (8.0 or more recent) 
  2. Jakarta Ant 

installation

  1. Téléchargez la distribution   application<version>-bin-.tar.gz
  2. Décompressez le contenu dans un dossier
  3. Ouvrez une denêtre de ligne de commande
  4. Tapez les commandes suivantes 
    5. %cd <intall directory>
%ant

Désinstallation

Supprimez le dossier d'installation

Test

%cd <install directory>
%cd bin
